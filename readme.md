# Introduction
ytl's Toolbox for opencv use, more introduction see toolbox.hpp

you can type
```
$ #include tlToolbox.hpp 
$ using namespace tl;
```
to use it


#Structure:
tlToolbox
--tlDebug

--tlGeometry
  --tlPoint
  --tlLine
  --tlPoly

--tlArray


其中
tlDebug为调试模块，用法类似QDebug
tlGeometry为计算几何模块，包含各种二维计算函数
tlArray为数组模块，包含一些多维数组的建立删除等操作
tlToolbox为opencv函数模块
